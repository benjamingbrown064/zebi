## Frequently Asked Questions

### What if we miss a deadline?
- Assess impact to launch (critical path or not?)
- Adjust subsequent deadlines if needed
- Document reason and learning
- Don't panic - focus on outcome, not date

### What if we discover new critical features?
- Add to backlog for post-launch
- Only add to v1 if absolutely essential for first customers
- Remember: Focus beats feature creep

### How do we know if beta customers are "active"?
- Daily login and task creation
- Using at least 3 core features
- Providing feedback
- Would pay if required

### What if we can't hit £5,000 MRR?
- Focus on beta customer success first
- Revenue is validation, not the only goal
- Lower target is acceptable if customers love product
- Better to have 5 paying customers who love it than 20 who don't
