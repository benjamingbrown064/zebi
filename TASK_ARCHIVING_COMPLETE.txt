================================================================================
                          SPECIFICATION COMPLIANCE
================================================================================

✅ Core Separation
   - Completion (completedAt) separate from archiving (archivedAt)
   - Completion = task is done
   - Archiving = task hidden from active views
   - Never delete data (soft archive only)

✅ Visibility Rules
   - Board: Hide archived, show active
   - List view: Toggle Active/Completed/Archived
   - Today: Exclude archived
   - Dashboard: Exclude archived

✅ Workspace Settings
   - Auto-archive retention days (0, 1, 7, custom)
   - Default: 7 days
   - Stored in Workspace model

✅ Manual Archive
   - Available in task overflow menu
   - Bulk action support
   - Sets archived_at = now()

✅ Auto-Archive Logic
   - Daily cron job at midnight UTC
   - Finds: completedAt IS NOT NULL AND archivedAt IS NULL 
           AND completedAt < now() - retention_period
   - Sets: archived_at = now()

✅ Goals & Progress
   - Goal calculations use completedAt only
   - Archived tasks ignored in progress
